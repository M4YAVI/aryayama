# Hono.js

Hono.js is a small, simple, and fast web framework for building modern web applications. It's designed to be lightweight while providing all the essential features needed for building robust web services.

## Overview

Hono (meaning "flame" in Japanese) is optimized for edge computing environments but works great anywhere JavaScript runs. It provides an Express-like API with modern features and TypeScript support out of the box.

## Key Features

- Ultralight & Fast
- TypeScript-first design
- Middleware support
- Built-in routing
- Edge-ready
- Compatible with various platforms (Cloudflare Workers, Deno, Bun, Node.js)

## Basic Usage

```typescript
import { Hono } from 'hono'

const app = new Hono()

// Basic routing
app.get('/', (c) => c.text('Hello Hono!'))

// JSON response
app.get('/api/users', (c) => {
  return c.json({
    users: [
      { id: 1, name: 'John' },
      { id: 2, name: 'Jane' }
    ]
  })
})

// Dynamic routes
app.get('/users/:id', (c) => {
  const id = c.req.param('id')
  return c.text(`User ID: ${id}`)
})

export default app
```

## Advanced Features

### Middleware

```typescript
import { Hono } from 'hono'
import { logger } from 'hono/logger'
import { cors } from 'hono/cors'

const app = new Hono()

// Add middleware
app.use('*', logger())
app.use('/api/*', cors())

// Custom middleware
app.use('*', async (c, next) => {
  console.time('response-time')
  await next()
  console.timeEnd('response-time')
})
```

### Validation

```typescript
import { Hono } from 'hono'
import { z } from 'zod'

const app = new Hono()

const userSchema = z.object({
  name: z.string(),
  email: z.string().email()
})

app.post('/users', async (c) => {
  const body = await c.req.json()
  const result = userSchema.safeParse(body)
  
  if (!result.success) {
    return c.json({ error: result.error }, 400)
  }
  
  return c.json({ user: result.data }, 201)
})
```
