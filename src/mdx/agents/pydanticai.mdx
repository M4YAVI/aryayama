# PydanticAI

PydanticAI is a powerful extension of Pydantic that adds AI-specific data validation and settings management capabilities using Python type annotations. It's designed to make working with AI models and data more robust and type-safe.

## Overview

PydanticAI enhances Pydantic's data validation capabilities with features specifically designed for AI applications, making it easier to handle complex AI model inputs and outputs.

## Key Features

- AI-specific data validators
- Type-safe model inputs and outputs
- Integration with popular AI frameworks
- Enhanced error handling for AI workflows
- Automatic data conversion and normalization

## Basic Usage

```python
from pydanticai import BaseAIModel
from pydantic import Field
from typing import List

class ModelInput(BaseAIModel):
    text: str = Field(..., description="Input text for the model")
    temperature: float = Field(0.7, ge=0.0, le=1.0)
    max_tokens: int = Field(100, gt=0)

class ModelOutput(BaseAIModel):
    generated_text: str
    confidence_score: float
    token_count: int

# Use the models
input_data = ModelInput(text="Hello, AI!", temperature=0.8)
output = model.generate(input_data)
validated_output = ModelOutput(**output)
```

## Best Practices

1. Define clear input and output models
2. Use appropriate field validators
3. Implement custom validators for complex AI logic
4. Handle edge cases and errors gracefully
5. Document model fields thoroughly

## Advanced Features

### Custom Validators

```python
from pydanticai import validator

class AIConfig(BaseAIModel):
    model_name: str
    batch_size: int

    @validator('batch_size')
    def validate_batch_size(cls, v):
        if not (v & (v-1) == 0):
            raise ValueError('Batch size must be a power of 2')
        return v
```

### Integration with AI Frameworks

```python
from pydanticai import AIModelConfig
from transformers import AutoModel

class TransformerConfig(AIModelConfig):
    model_name: str
    num_attention_heads: int
    hidden_size: int

    def load_model(self):
        return AutoModel.from_pretrained(self.model_name)
```
